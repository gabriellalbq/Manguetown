# Manguetown

<p>
  O jogo Manguetown: do Recife ao caos é um projeto de jogo físico atrelado a um arduíno com o propósito de incluir pessoas com dislexia e/ou deficiência cognitiva no universo de jogos.
  <br>
  <br>
  Este é o código que enviamos ao nosso arduino para que ele possa realizar as funções que desejamos e permitir a execução do jogo, sendo elas:
</p>

<li>Rodar os dados;</li>
<li>Indicar a quantiade de vidas de cada jogador;</li>
<li>Indicar a quantidade de vidas de cada jogador;</li>
<li>Avisar eliminações e o vencedor no fim do Jogo.</li>
